### Prerequisite
Ensure you have the following installed:
 1. kotlin
 2. Java

#### Setup
``` 
git clone https://github.com/Wambug/learn_kotlin
```
#### Run
```
kotlinc main.kt -include-runtime -d main.jar
kotlin -classpath main.jar MainKt.class 
```
